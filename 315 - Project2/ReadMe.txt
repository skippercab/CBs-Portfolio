This program is a simple HTML parser that runs inside of the command terminal
and tells you whether or not your HTML tags are balanced.

To compile this code, go to terminal and navigate to this project folder
(the one titled "HTML Parser") and type the following:
$ make

To run this code, stay in the same folder and type the following:
$ ./html-test "(whatever file path you wish)"
