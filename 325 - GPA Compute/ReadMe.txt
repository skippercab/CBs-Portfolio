This project computes a GPA from a selected text file and the grades inside.

To compile this code, go to the command terminal and type the following:
$ cd "(path to Compute_GPA_From_File.java)"
$ javac Compute_GPA_From_File.java

To run this code, stay in the same command terminal session and type the
following:
$ java Compute_GPA_From_File
